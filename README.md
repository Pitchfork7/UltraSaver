# UltraSaver
Ultra Kill Save Manager

## How it works
I have no clue if this works due to being dead tired while coding this. To use this program put the py file or exe(if i make one) into the same folder as the folder with the ultrakill save file. When wanting to switch files you first load the real file and save it to whatever file you want (ex. save1). Then you load your fake file(ex. save2) and save it to real file. Now the game will load the new file. Do the same to switch back.
